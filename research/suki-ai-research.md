## 2. Product Offerings

### Suki Assistant
The flagship AI-powered voice assistant for clinicians featuring:

**Core Capabilities:**
- **Ambient Clinical Documentation:** Automatically generates clinical notes by listening to patient-clinician conversations
- **Voice-Enabled Editing:** Hands-free note modification and refinement
- **Problem-Based Charting:** Flexible documentation adapting to clinician workflows
- **Clinical Q&A:** Answers clinical questions in real-time
- **Patient Summarization:** Generates patient summaries from EHR data

**Advanced Features:**
- **Ambient Orders Staging (April 2025):** Industry-first capability to generate and stage prescription orders from ambient visits
  - Extracts order details from conversation
  - Structures orders with correct sig, quantity, refills, frequency, duration
  - Codes orders accurately
  - Stages in EHR for clinician review and approval
  - Expanding to labs, imaging, referrals across major EHRs

**Coding Capabilities:**
- ICD-10 code generation
- HCC (Hierarchical Condition Category) coding
- CPT code suggestions
- E/M (Evaluation & Management) level recommendations

**Platform Support:**
- Web application
- Native Windows application
- Mobile app (iOS and Android)
- 100+ medical specialties supported

### Suki Platform
Developer tools enabling technology partners to embed Suki's AI capabilities:

**Components:**
- SDK (Software Development Kit)
- APIs for voice AI integration
- White-label ambient AI capabilities

**Platform Partners:**
- athenahealth (integrated SDK into athenaOne)
- MEDENT
- Azalea Health
- MEDITECH (Alliance Partner)

### Suki Compose
Ambient clinical documentation solution specifically for:
- Note generation from patient encounters
- Integration with existing EHR workflows

### Suki for Nurses (October 2025)
New consortium-developed product for nursing workflows:
- Consortium members: McLeod Health, Citizens Memorial, Boone Health, Fisher-Titus
- Partnership with AvaSure (AI virtual care platform serving 1,100+ hospitals)
- Integrations with Epic, MEDITECH, Oracle Health

## 6. Technology Stack

### Core Architecture

**Speech Recognition (ASR):**
- Medically-tuned ASR with industry-leading accuracy
- Advanced voice recognition comparable to Siri/Alexa
- Accent-agnostic (works with international clinicians)

**Natural Language Understanding (NLU):**
- Enables natural speech (no rote commands required)
- Contextual understanding of clinical conversations

**Large Language Model (LLM):**
- Medically-optimized LLM
- Trained on structured data from 1M+ patient records
- Covers 35+ medical specialties
- Proprietary model (specific architecture not disclosed)

### Integration Architecture
- **Bidirectional EHR Sync:** Real-time data flow with EHRs
- **API/SDK Platform:** Enables third-party integration
- **Cloud-Based:** Scalable infrastructure

### Platform Capabilities
- Real-time ambient listening
- Generative AI note creation
- Voice-enabled commands
- Automated coding suggestions
- Order staging and structuring
