## How to Use These Prompts

1. **Open Claude Code Research Preview** (web version)
2. **Copy one complete prompt** (everything between the ``` marks)
3. **Paste and run**
4. **Save output** to `/vendors/[vendor-name].md` in the GitHub repo
5. **Check off** in the vendor list

## Batch Processing Tips

- Run 3-5 agents in parallel (separate browser tabs)
- Save outputs immediately to avoid losing research
- Note any gaps for follow-up research
- Update the vendor comparison matrix after each batch
